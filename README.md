## Requirements

- Node.js 18+
- npm 9+

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

> The app is designed for a 390×844 phone viewport. For the best experience, use browser DevTools in responsive mode or view on a mobile device.
